## 1. 界面布局

```
┌──────────────────────────────────────────────────┐
│  TitleBar（自定义标题栏）                            │
├──────────────────────────────────────────────────┤
│  Toolbar（后退/前进/向上/刷新 | 地址栏 | 搜索框）      │
├──────────────────────────────────────────────────┤
│  RibbonToolbar（新建文件夹/新建文件/剪切/复制/粘贴/删除/属性）│
├────────┬─────────────────────┬──────────────────┤
│        │                     │                  │
│ Sidebar│    PanesArea        │  PropertiesPanel │
│ ─────  │  ┌────────────────┐ │  ┌──────────────┐│
│ 此电脑  │  │ Tab1 │ Tab2 │+│ │  │ 预览区域      ││
│ 驱动器  │  ├────────────────┤ │  │              ││
│ (C:)   │  │ Breadcrumb     │ │  │ ──────────── ││
│ (D:)   │  ├────────────────┤ │  │ 属性信息      ││
│ ─────  │  │                │ │  │ 文件名/路径   ││
│ 快速访问│  │ FileList       │ │  │ 大小/日期    ││
│ 收藏夹  │  │ 详情/列表/网格 │ │  │              ││
│        │  │ 树形/分栏       │ │  └──────────────┘│
│        │  └────────────────┘ │                  │
├────────┴─────────────────────┴──────────────────┤
│  StatusBar（状态信息）                              │
└──────────────────────────────────────────────────┘
```

### 区域说明

| 区域 | 功能 |
|------|------|
| **TitleBar** | 窗口标题栏，支持拖拽移动窗口。macOS 显示交通灯按钮 |
| **Toolbar** | 导航按钮 + 地址栏（支持路径自动补全）+ 搜索框（带历史记录）|
| **RibbonToolbar** | 常用文件操作快捷按钮 |
| **Sidebar** | 左侧导航面板：此电脑、驱动器、快速访问、收藏夹 |
| **PanesArea** | 核心文件浏览区域，支持多标签页和分屏 |
| **PropertiesPanel** | 右侧预览/属性面板，可拖拽调整宽度 |
| **StatusBar** | 底部状态栏，显示当前路径和文件数 |

## 2. 文件浏览

### 2.1 5 种视图模式

| 视图 | 快捷键 | 说明 |
|------|--------|------|
| **详细信息 (Details)** | — | 表格形式，显示名称/修改日期/创建日期/类型/大小，列宽可拖拽 |
| **列表 (List)** | — | 紧凑列表，适合大量文件快速扫描 |
| **网格 (Grid)** | — | 图标模式，显示文件缩略图和扩展名彩色图标 |
| **树形 (Tree)** | — | 左侧树 + 右侧内容，适合目录深度导航 |
| **分栏 (Column)** | — | Miller Columns，多列连续下钻，macOS Finder 风格 |

### 2.2 地址栏

- 显示当前完整路径
- 输入路径时自动补全子目录名
- 支持 `C:\`、`C:/Users`、`/home/user` 等跨平台路径格式
- 按 `Enter` 导航到输入路径

### 2.3 面包屑导航

- 路径按分段显示，点击任一段跳转到该目录
- Windows 盘符自动识别（`C:\`、`D:\`）
- Unix 根路径（`/`）正确处理

### 2.4 导航历史

- 工具栏后退/前进按钮
- 快捷键：`Alt+←` 后退、`Alt+→` 前进
- 最多保存 50 条历史

## 3. 文件操作

### 3.1 基础操作

| 操作 | 方法 | 快捷键 |
|------|------|--------|
| 打开文件/进入目录 | 双击 / 选中后 Enter | `Enter` / `Space` |
| 返回上级目录 | 工具栏向上按钮 | `←` / `Backspace` |
| 新建文件夹 | 右键 → 新建文件夹 / Ribbon 按钮 | `Ctrl+Shift+N` |
| 新建文件 | 右键 → 新建文件 / Ribbon 按钮 | `Ctrl+N` |
| 重命名 | 右键 → 重命名 | `F2` |
| 删除 → 回收站 | 右键 → 删除 | `Delete` |
| 永久删除 | 右键 → 删除 → 永久删除 | `Shift+Delete` |
| 刷新 | 右键 → 刷新 | `F5` |

### 3.2 剪贴板操作

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 剪切 | `Ctrl+X` | 文件标记为剪切状态，粘贴后移动 |
| 复制 | `Ctrl+C` | 复制到内部剪贴板 |
| 粘贴 | `Ctrl+V` | 粘贴到当前目录 |
| 复制路径 | 右键 → 复制路径 | 多文件换行分隔 |
| 取消剪切 | `Escape` | 取消剪切状态 |

### 3.3 压缩/解压

- **压缩**：选中文件/文件夹 → 右键 → 压缩 → 选择 ZIP 保存路径
- **解压**：选中 ZIP/TAR/TAR.GZ/7z/RAR → 右键 → 解压 → 选择目标目录
- 解压时自动防护 ZipSlip 攻击（拒绝 `..` 和绝对路径）

### 3.4 拖放操作

- 文件/文件夹拖入其他目录移动
- 拖拽标签页切换
- 支持拖出到外部应用（Windows QQ/微信/Chrome 等）

## 5. 预览功能

### 5.1 右侧属性面板

选中文件后，右侧面板自动显示预览：

| 文件类型 | 预览能力 |
|---------|----------|
| **图片** | 缩放、旋转（90°/180°/270°）、拖拽平移、鼠标滚轮缩放 |
| **PDF** | 缩放、拖拽平移、滚轮缩放 |
| **Office 文档** | docx / xlsx / pptx 内联渲染 |
| **视频** | DPlayer 播放器、截图、倍速、播放位置记忆 |
| **代码** | CodeMirror 5 代码高亮、50+ 语言 |
| **Markdown** | 编辑/预览切换、Shiki 代码高亮、导出 HTML/Word/PDF |
| **文本** | 纯文本显示 |
| **归档** | ZIP/7z/RAR/TAR 内容列表浏览 |
| **二进制/不支持** | 提示「没有预览」，提供外部程序打开按钮 |

### 5.2 独立预览窗口

点击属性面板右上角的弹出按钮，打开独立预览窗口：

```
┌──────────────────────────────────────────────┐
│  TitleBar（拖拽移动）   重命名输入框            │
├──────────────────────────────────────────────┤
│  Toolbar: 打开 / 重命名 / 另存为 / 删除 /     │
│           复制路径 / 在资源管理器显示           │
├──────────┬───────────────────────────────────┤
│          │  ZoomBar（缩放/旋转/重置）           │
│  左侧树   ├───────────────────────────────────┤
│  文件导航  │  预览内容                          │
│          │  图片/PDF/Office/视频/代码/文本     │
│          │                                   │
│          │                                   │
├──────────┴───────────────────────────────────┤
│  右键菜单: 打开 / 在资源管理器显示 / 复制路径     │
└──────────────────────────────────────────────┘
```

独立预览窗口特点：
- 自适应窗口尺寸（约主窗口 65%×75%）
- 左侧文件树导航，支持键盘操作
- 工具栏：打开外部程序、重命名、另存为、删除、复制路径、在资源管理器显示
- 独立于主窗口，可同时打开多个

## 6. 搜索

### 6.1 基本搜索

在工具栏搜索框中输入关键词，按 `Enter` 开始搜索。

### 6.2 通配符搜索

| 通配符 | 说明 | 示例 |
|--------|------|------|
| `*` | 匹配任意字符 | `*.rs` 匹配所有 Rust 文件 |
| `?` | 匹配单个字符 | `report?.pdf` 匹配 report1.pdf |
| `\|` | OR 条件 | `*.vue \| *.ts` 匹配 Vue 或 TS 文件 |

### 6.3 大小过滤

| 示例 | 说明 |
|------|------|
| `>10MB` | 大于 10MB 的文件 |
| `<1KB` | 小于 1KB 的文件 |
| `>100MB \| *.zip` | 大于 100MB 或 ZIP 文件 |

### 6.4 搜索历史

- 自动保存最近 15 条搜索记录
- 搜索框聚焦时弹出历史下拉
- ↑↓ 选择，Enter 确认
- 支持清除全部历史

## 7. 收藏夹

### 7.1 添加收藏

- 右键文件夹 →「添加到收藏夹」
- 已收藏的文件夹显示「从收藏夹移除」

### 7.2 移除收藏

- 侧栏书签右键 → 确认对话框 → 移除
- 右键已收藏的文件夹 →「从收藏夹移除」

### 7.3 导航

- 单击侧栏书签直接跳转到该文件夹
- 当前所在目录的书签自动高亮

## 8. 系统托盘

### 8.1 托盘菜单

```
 显示主窗口 / 隐藏主窗口（动态切换）
 ————————————
 下载
 文档
 ————————————
 清理缓存数据
 ————————————
 设置…
 ————————————
 退出
```

### 8.2 托盘行为

| 操作 | 行为 |
|------|------|
| 左键单击 | 切换窗口显隐 |
| 左键双击 | 强制显示并聚焦 |
| 关闭窗口（关闭不退出） | 隐藏到托盘 |
| 关闭窗口（关闭时退出） | 直接退出 |
| 菜单→退出 | 自动保存会话后退出 |

### 8.3 设置选项

- **显示系统托盘**：开启/关闭托盘图标
- **关闭界面时退出**：开启后关闭窗口直接退出；关闭则隐藏到托盘
