## 1. 界面布局

```
┌──────────────────────────────────────────────────┐
│  TitleBar（自定义标题栏）                            │
├──────────────────────────────────────────────────┤
│  Toolbar（← → ↑ ↻ | 地址栏 | 搜索）                 │
├──────────────────────────────────────────────────┤
│  RibbonToolbar（新建/剪切/复制/粘贴/删除）            │
├────────┬─────────────────────┬──────────────────┤
│ Sidebar│    PanesArea        │  PropertiesPanel │
│ ─────  │  ┌────────────────┐ │  ┌──────────────┐│
│ 此电脑  │  │ Tab1 │ Tab2 │+│ │  │ 预览区域      ││
│ 驱动器  │  ├────────────────┤ │  │              ││
│ 快速访问│  │ Breadcrumb     │ │  │ ──────────── ││
│ 收藏夹  │  ├────────────────┤ │  │ 属性信息      ││
│        │  │ FileList       │ │  │              ││
│        │  └────────────────┘ │  └──────────────┘│
├────────┴─────────────────────┴──────────────────┤
│  ┌─ Terminal Panel ─ Ctrl+` ──────────────────┐  │
│  │ ▸ zsh ｜ ~/Documents  ● 13px [−][+][⊠][⟳][✕]│  │
│  │ apple@mac ~/D $ █                          │  │
│  └────────────────────────────────────────────┘  │
├──────────────────────────────────────────────────┤
│  StatusBar（/path  [>_] [▯│] [N items]）          │
└──────────────────────────────────────────────────┘
```

### 区域说明

| 区域 | 功能 |
|------|------|
| **TitleBar** | 窗口标题栏，拖拽移动窗口 |
| **Toolbar** | 导航按钮 + 地址栏（自动补全）+ 搜索框（历史记录） |
| **RibbonToolbar** | 常用文件操作快捷按钮 |
| **Sidebar** | 此电脑、驱动器、快速访问、收藏夹 |
| **PanesArea** | 核心文件浏览区域：多标签页 + 分屏 |
| **PropertiesPanel** | 右侧预览/属性面板，可拖拽调整宽度 |
| **Terminal Panel** | 底部终端面板，可折叠/最大化/拖拽高度 |
| **StatusBar** | 当前路径 + 终端/属性切换 + 文件数 |

## 2. 文件浏览

### 2.1 5 种视图模式

| 视图 | 说明 |
|------|------|
| **详细信息** | 表格：名称/修改日期/创建日期/类型/大小，列宽可拖拽 |
| **列表** | 紧凑列表，适合大量文件快速扫描 |
| **网格** | 图标模式，缩略图和彩色图标 |
| **树形** | 左侧树 + 右侧内容，深度导航 |
| **分栏** | Miller Columns，多列下钻，macOS Finder 风格 |

### 2.2 地址栏

- 显示当前完整路径
- 输入路径时自动补全子目录名
- 支持 `C:\`、`C:/Users`、`/home/user` 等跨平台格式

### 2.3 面包屑导航

- 路径分段显示，点击任一段跳转到该目录
- Windows 盘符 (`C:`) 和 Unix 根 (`/`) 自动识别

### 2.4 导航历史

- 后退/前进/向上按钮
- `Alt+←` / `Alt+→` 快捷键
- 最多 50 条历史

## 3. 文件操作

### 3.1 基础操作

| 操作 | 方法 | 快捷键 |
|------|------|--------|
| 打开 | 双击 / Enter | `Enter` / `Space` |
| 返回上级 | 向上按钮 | `←` / `Backspace` |
| 新建文件夹 | 右键 / Ribbon | `Ctrl+Shift+N` |
| 新建文件 | 右键 / Ribbon | `Ctrl+N` |
| 重命名 | 右键 | `F2` |
| 删除 | 右键 → 删除 | `Delete` |
| 永久删除 | — | `Shift+Delete` |
| 刷新 | — | `F5` |

### 3.2 剪贴板

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 剪切 | `Ctrl+X` | 标记剪切，粘贴后移动 |
| 复制 | `Ctrl+C` | 复制到内部剪贴板 |
| 粘贴 | `Ctrl+V` | 粘贴到当前目录 |
| 复制路径 | 右键 | 多文件换行分隔 |
| 取消剪切 | `Escape` | — |

### 3.3 压缩/解压

- **压缩**：选中文件 → 右键 → 压缩 → ZIP
- **解压**：选中压缩包 → 右键 → 解压 → 选择目录
- 支持 ZIP/TAR/GZ/BZ2/XZ/7z/RAR
- 自动 ZipSlip 防护

### 3.4 拖放

- 文件拖入目录移动
- 拖拽标签页切换
- Windows 拖出到 QQ/微信/Chrome 等外部应用

## 6. 内置终端

### 6.1 打开/关闭

| 方式 | 说明 |
|------|------|
| `` Ctrl+` `` | 全局快捷键切换 |
| StatusBar `>_` | 点击按钮切换 |
| Panel `✕` | 关闭面板 |

### 6.2 面板元素

```
┌─ ═══ 拖拽 ═══ ──────────────────────────────────┐
│  ▸ zsh ｜ ~/Documents    ● 13px [−][+][⊠][⟳][✕]│
├──────────────────────────────────────────────────┤
│  apple@mac ~/D $ █                              │
└──────────────────────────────────────────────────┘
```

| 元素 | 说明 |
|------|------|
| `▸ zsh` | 当前 shell（macOS zsh / Linux bash / Win PowerShell） |
| `｜路径` | 当前工作目录 |
| `●` | 绿=运行 / 黄=启动 / 红=退出 |
| `13px` | 字体大小 |
| `[−][+]` | 缩小/放大字体 |
| `[⊠]`/`[🗗]` | 最大化/还原 |
| `[⟳]` | 重启终端 |
| `[✕]` | 关闭面板 |

### 6.3 终端键盘

| 按键 | 行为 |
|------|------|
| 普通输入 | 传递给 shell |
| `Ctrl+C`（无选中） | 发送 SIGINT 中断命令 |
| `Ctrl+C`（有选中） | 复制选中文本 |
| `Ctrl+V` | 粘贴剪贴板 |

### 6.4 面板操作

| 操作 | 方式 |
|------|------|
| 最大化 | `⊠` / 双击 header |
| 还原 | `🗗` / `Esc` / 双击 header / StatusBar 指示条 |
| 调高度 | 拖拽顶部边缘 |
| 调字体 | `Ctrl+=`/`Ctrl+-`/`Ctrl+0` 或 `+`/`−` 按钮 |

### 6.5 目录跟随

文件浏览切换目录 → 终端自动 `cd` 到新路径。

### 6.6 退出恢复

进程退出后显示覆盖层，点击 `⟳` 或「重启终端」恢复。

### 6.7 主题配色

终端配色跟随应用主题自动切换，7 套均含完整 ANSI 16 色：

| 主题 | 背景 |
|------|------|
| Catppuccin 暗 | `#1e1e2e` |
| Catppuccin 亮 | `#eff1f5` |
| Nord | `#2e3440` |
| Tokyo Night | `#1a1b26` |
| One Dark Pro | `#282c34` |
| Dracula | `#282a36` |
| Solarized 亮 | `#fdf6e3` |

### 6.8 跨平台 Shell

| 平台 | 默认 |
|------|------|
| macOS | `$SHELL` / `/bin/zsh` |
| Linux | `$SHELL` / `/bin/bash` |
| Windows | `%COMSPEC%` / `powershell.exe` |

## 7. 搜索

### 7.1 基本搜索

工具栏搜索框输入，`Enter` 开始。

### 7.2 通配符

| 符号 | 说明 | 示例 |
|------|------|------|
| `*` | 任意字符 | `*.rs` |
| `?` | 单个字符 | `report?.pdf` |
| `\|` | OR | `*.vue \| *.ts` |

### 7.3 大小过滤

| 示例 | 说明 |
|------|------|
| `>10MB` | 大于 10MB |
| `<1KB` | 小于 1KB |

### 7.4 搜索历史

- 最近 15 条持久化
- ↑↓ 选择，Enter 确认
- 支持清除全部

## 10. 键盘快捷键

### 10.1 文件列表

| 键 | 功能 |
|----|------|
| `↑`/`↓` | 移动焦点 |
| `→` | 进入目录 |
| `←` | 返回上级 |
| `Home`/`End` | 首/尾 |
| `PageUp`/`PageDown` | 翻页 |
| `Shift+↑/↓` | 范围选中 |

### 10.2 全局

| 快捷键 | 功能 |
|--------|------|
| `Enter`/`Space` | 打开 |
| `Ctrl+C`/`X`/`V`/`A` | 复制/剪切/粘贴/全选 |
| `Delete`/`Shift+Delete` | 删除/永久删除 |
| `F2` | 重命名 |
| `F5` | 刷新 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+W` | 关闭标签 |
| `Ctrl+Tab` | 切换标签 |
| `Ctrl+N`/`Ctrl+Shift+N` | 新建文件/文件夹 |
| `Ctrl+P` | 属性面板 |
| `Ctrl+,` | 设置 |
| `Backspace` | 返回上级 |
| `Escape` | 取消剪切/终端还原/关闭弹窗 |

### 10.3 终端专用

| 快捷键 | 功能 |
|--------|------|
| `` Ctrl+` `` | 切换终端面板 |
